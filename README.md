# Study with me!

# Features
昨年からオンライン授業が始まり、家で一人で勉強する時間が増えました。その結果、友達となかなか話せないといった問題を私の友達が抱えていました。そこで私は、友達を助けるためのアプリを作成しようと考えました。<br>
友達と同じタイミングで勉強や休憩をし、休憩中のみ雑談をするというアイデアを考え、それを可能にするAndroidアプリを作成しました。<br>
